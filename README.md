# AI Basics Course - Tilburg International Center

Welcome to the AI Basics Course! This course is designed to provide the theoretical and practical foundations for interacting with modern language models.

## Course Structure

The course consists of **4 sessions**, each lasting **1.5 hours**.

| Session | Title | Description |
| :--- | :--- | :--- |
| 1 | **Basics & History** | AI history, keywords, main providers, and introductory video. |
| 2 | **Talking to AI** | Guided practice: hallucinations, limits, multi-model use, and interfaces (Online vs IDE). |
| 3 | **Grand Laboratory (P1)** | Project selection, outlining, and initial work. |
| 4 | **Grand Laboratory (P2)** | Project completion and final reflections. |

## Venue

**International Center of Tilburg**

## Objectives

- Understand what AI is and how it has evolved.
- Learn how to communicate effectively with models (Prompt Engineering).
- Experiment with different tools beyond the classic chat interface.
- Complete a small, concrete project.
